The 4 networks for the MNIST data set are presented in this folder.
Each sub folder contains:
	> Netron Models: The collection of all netron models for all types of folding. View each netron model using the netron app.
	> A zip file: The zip file of the least folded network. This is to be deployed to the fpga.
	> More Folded Networks: The folder contains the other networks that had a higher degree of folding applied to them.
